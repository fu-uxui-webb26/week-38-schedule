## Introduktion

Hur vet vi om en design faktiskt fungerar? Och hur påverkar visuella designval användarupplevelsen?

Den här veckan fördjupar vi oss i prototypframtagning, användbarhet och tillgänglighet. Vi fortsätter utveckla våra lösningar i Figma samtidigt som vi arbetar med färg, typografi, layout och visuell hierarki.

Vi introducerar även grundläggande principer för WCAG och diskuterar hur små designbeslut kan göra stor skillnad för användbarheten.

Målet är att förvandla våra tidiga idéer till genomtänkta och användarvänliga gränssnitt.
